# Synopsis
Python Number Guessing Game

## Description
A random number guessing game where user guess a number and the program will tell whether if that number is lower or higher than the correct number until user gets the right number.

## Contents
README - The manual file
guess.py - Implementation of Python Number Guessing Game

## Compilation
python3 guess.py

## Execution Instruction
When prompted "Enter your guess: ", enter a number to start playing

## Dependencies
Python3 - Programming language to execute
random - To generate a random number
